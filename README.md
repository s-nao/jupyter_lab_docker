## コマンド

##### 起動
```
docker-compose up
```

##### 停止
```
Ctrl+c
```

## 動作の確認
ブラウザで、
```
http://localhost:1088
```
を検索する

## 使用ミドルウェア
・docker version 17.09.1-ce

・docker-compose version 1.17.1
